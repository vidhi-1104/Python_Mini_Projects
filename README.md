# Rock, Paper, Scissors (Python)

A simple command-line version of the classic game.

## How to Play
1. Click the [Repl.it Link] or [Binder Link] below.
2. Type `0` for Rock, `1` for Paper, or `2` for Scissors.
3. See if you can beat the computer!

## Logic
- Rock beats Scissors.
- Scissors beats Paper.
- Paper beats Rock.
